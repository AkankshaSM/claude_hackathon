# Claude Hackathon - Best for College Students

> A powerful AI-powered tool designed specifically for college students to enhance their academic journey

## Overview

This project was created for the Claude Hackathon with the theme "Best for College Students". It leverages Claude's advanced AI capabilities to solve real challenges that college students face daily, from managing coursework to career planning.

## Problem Statement

College students today face numerous challenges:
- **Information Overload**: Difficulty managing multiple courses, assignments, and deadlines
- **Study Efficiency**: Struggling to study effectively and retain information
- **Career Preparation**: Navigating internships, job applications, and skill development
- **Time Management**: Balancing academics, extracurriculars, and personal life
- **Financial Constraints**: Limited budget for expensive tools and resources

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/claude_hackathon.git
cd claude_hackathon

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Claude API key to .env

# Run the application
npm start
```

### Configuration

1. Obtain a Claude API key from [Anthropic](https://www.anthropic.com)
2. Add your API key to the `.env` file:
   ```
   CLAUDE_API_KEY=your_api_key_here
   ```
3. Configure any additional settings in `config.js`
